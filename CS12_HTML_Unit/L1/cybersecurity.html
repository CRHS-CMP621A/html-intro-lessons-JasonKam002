Cyber Security and HTML: 5 Historical Cases

1. Google China Attack (2010)

Hackers targeted Gmail accounts of activists using a "sophisticated" attack that began with a malicious link, launching malicious JavaScript code that took advantage of a Microsoft Internet Explorer vulnerability to breach systems.

2. Target and Home Depot (2013-2014)

Hackers stole credit card data from these retailers, in part by exploiting web-based vulnerabilities, with data eventually surfacing on a Ukrainian cybercrime site.

3. First American Financial Corp. (2019)

An Insecure Direct Object Reference (IDOR) vulnerability allowed access to 885 million files. The flaw meant that anyone with a link could view sensitive documents by simply altering a number in the URL.

4. Adobe Breach (2013)

Attackers not only obtained user records but also over 40 GB of source code for Adobe products (Acrobat, ColdFusion), highlighting risks to both data and application integrity.

5. Ashley Madison Hack (2015)

The Impact Team breached the dating site, stealing user data and subsequently dumping 9.7 gigabytes of user information.
